6 security checks

The app runs six real-time tests to assess how secure your Tor connection actually is.

🔐

Orbot Status

Verifies that the SOCKS5 proxy is active on 127.0.0.1:9050, distinguishing between Orbot VPN, third-party VPN, and no VPN.

● Proxy active
🌐

IP via Tor Proxy

Compares your Tor exit IP against your real IP. Shows geolocation (flag, city, country) and latency in ms for both.

● Different IPs ✓
🔍

DNS Leak Test

Checks that DNS requests don't exit outside the Tor network, which could expose your location or browsing habits.

● No leak ✓
✅

TorProject Check

Official check via check.torproject.org/api/ip — the authoritative source to confirm Tor usage.

● Using Tor ✓
🧅

.onion Address

Tests reachability of any custom .onion address with a 30-second timeout for dark web sites.

● Online / Offline
📡

WebRTC Leak Test

Sends a direct UDP STUN packet (bypassing proxy) to Cloudflare and stunprotocol.org to detect IP leaks.

⚠ UDP exposed

Connection modes

Due modalità, quattro scenari

Seleziona la modalità e osserva come cambia la copertura combinata con lo stato della VPN Orbot.

Tor Proxy VPN off

HTTP via SOCKS5 (9050). UDP non protetto — leak WebRTC possibile.

Tor Proxy VPN on

Doppia copertura Tor. UDP bloccato dalla VPN — nessun leak possibile.

⭐ Più sicuro

Diretto VPN on

TCP instradato via Tor a livello OS. UDP bloccato dalla VPN Orbot.

Diretto VPN off

Nessuna protezione — utile come baseline per vedere il tuo IP reale.

Two modes, four scenarios

Select a mode and see how coverage changes combined with the Orbot VPN state.

Tor Proxy VPN off

HTTP via SOCKS5 (9050). UDP unprotected — WebRTC leak possible.

Tor Proxy VPN on

Dual Tor coverage. UDP blocked by VPN — no leak possible.

⭐ Most secure

Direct VPN on

TCP routed through Tor at OS level. UDP blocked by Orbot VPN.

Direct VPN off

No protection — useful as a baseline to see your real IP.
